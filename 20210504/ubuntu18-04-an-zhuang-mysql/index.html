Ubuntu18.04 安装mysql

MySQL
Linux Tools
发布日期:   2021-05-04
更新日期:   2021-05-04
文章字数:   203
阅读时长:   1 分
阅读次数:  

1. 安装MySQL

sudo apt update

sudo apt install mysql-server  # mysql-5.7

2. 配置MySQL

sudo mysql_secure_installation

这一步的作用是:

  • 配置root密码
  • 配置root是否可远程访问
  • … (其他基本配置)

3. 访问MySQL

# 通过root用户访问mysql是通过auth_socket插件验证而非通过密码验证
$ sudo mysql

# 普通用户访问mysql需要输入用户名和密码通过密码验证
$ mysql -u root -p

4. 创建用户或者修改密码

# mysql 验证插件: caching_sha2_password
mysql> CREATE USER 'USER'@'HOST' IDENTIFIED BY 'PASSWD';
mysql> ALTER USER 'USER'@'HOST' IDENTIFIED BY 'PASSWD';

# 如果上面那条指令报错, 说明mysql验证密码的方式是另一种插件: mysql_native_password
mysql> CREATE USER 'USER'@'HOST' IDENTIFIED WITH mysql_native_password BY 'PASSWORD';
# 在这种插件下, 修改密码的方式也不一样:
mysql> ALTER USER 'USER'@'HOST' IDENTIFIED WITH mysql_native_password BY 'PASSWORD';
